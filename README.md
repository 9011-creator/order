#### Silver Hospital AI
### 노인을 위한 병원 추천 및 3D 길 안내 시뮬레이터
## 1. 프로젝트 개요
Silver Hospital AI는 고령자가 자신의 증상과 이동 방법에 맞는 병원을 추천받고, 병원까지 이동하는 과정을 간단한 3D 환경에서 체험할 수 있도록 제작한 웹 기반 프로그램이다.

사용자는 나이, 불편한 부위, 이동 방식, 우선 기준을 선택할 수 있으며, 선택된 정보를 바탕으로 추천 병원과 추천 이유가 제공된다.

추천 결과 확인 후 3D 체험 모드를 실행하면 병원까지 직접 이동해 볼 수 있다.

# 실행 환경
## 2. 문제 정의
# 이동약자
- 고령자
- 휠체어 이용자
- 병원을 처음 방문하는 사람

# 문제 상황
고령자는 병원을 방문할 때 다음과 같은 어려움을 겪을 수 있다.
- 어떤 병원을 방문해야 하는지 판단하기 어렵다.
- 이동 경로를 이해하기 어렵다.
- 계단, 횡단보도 등 이동 과정에서 불편함이 발생할 수 있다.
- 병원 위치를 찾는 과정에서 혼란을 겪을 수 있다.

# 해결 목표
사용자의 상태에 적합한 병원을 추천하고, 병원까지 이동하는 과정을 가상으로 체험할 수 있도록 하여 길 찾기에 대한 부담을 줄인다.

## 3. 솔루션 정의
프로그램은 사용자의 선택 정보를 입력받아 적절한 병원을 추천한다.
추천 결과에는 다음 정보가 포함된다.
- 추천 병원
- 추천 점수
- 추천 이유
- 예상 거리
 예상 이동 시간

이후 3D 체험 기능을 통해 사용자는 추천된 병원까지 직접 이동할 수 있다.

## 4. 주요 기능
# 병원 추천 기능
선택 항목( 나이, 불편한 곳, 이동 방법 등)을 제공하여 자신의 현 상황을 판단한다.
선택 항목에 따라서 추천 병원, 추천 점수, 추천 이유가 출력된다.

# 점수 계산
기본 점수 : 80점
추가 점수
- 70대 이상 : +5점
- 무릎 또는 허리 : +5점
- 걷기 어려움 : +5점
- 휠체어 이용 : +10점

# 추천 이유 생성
입력 조건에 따라 다음과 같은 이유를 표시한다.
- 고령자 맞춤 진료 가능
- 재활 치료 시설 보유
- 휴식 공간이 많은 경로
- 휠체어 이동 가능
- 계단 없는 안전 경로
- 최단 거리 경로

## 5. 시스템 작동 구조
# Input
- 나이
- 증상
- 이동 방식
- 우선 기준

# Process
1. 사용자 정보 입력
2. 조건에 따른 병원 선택
3. 추천 점수 계산
4. 추천 이유 생성
5. 결과 화면 출력
6. 3D 환경 생성
7. 병원까지 이동 체험

# Output
- 추천 병원
- 추천 점수
- 추천 이유
- 이동 체험
- 병원 도착 안내

## 6. 3D 시뮬레이터 구성
1. 구성 요소
2. 이동 시스템
 - 조작 방법
    W : 전진
    S : 후진
    A : 왼쪽 이동
    D : 오른쪽 이동
    마우스 : 시점 변경
 - 이동 규칙
   사용자는 1인칭 시점으로 이동한다.
   현재 위치는 미니맵에 표시된다.
3. 신호등 시스템
- 빨간불
 횡단보도 통과 불가
- 초록불
 횡단보도 통과 가능

신호는 일정 시간마다 자동으로 변경된다

4. 병원 도착 기능

사용자가 병원 근처까지 이동하면 도착으로 판정한다.
도착 시
병원 도착 메시지 출력
이동 종료
초기 화면 복귀
기능을 수행한다.

5. 기대 효과
노인의 병원 선택을 지원할 수 있다.
이동약자를 고려한 서비스 설계를 경험할 수 있다.
3D 시뮬레이션을 통해 경로 안내 기능을 체험할 수 있다.
웹 기반으로 누구나 쉽게 사용할 수 있다.

6. 한계점
현재 프로젝트는 실제 병원 데이터와 연동되지 않는다.
또한 실제 지도 서비스나 GPS를 사용하지 않고 가상의 3D 공간에서 동작한다.
따라서 실제 길 안내 서비스라기보다 병원 추천 및 이동 체험을 위한 개념 검증(Prototype) 수준의 시뮬레이터이다.
